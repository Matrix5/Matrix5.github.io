Java tunnistused

Iga õige javanisti kodus ilutseb Meie püharaamat - selleks on Java kirjutatud pühakirjad. See kogum kannab nime "Java tunnistused" ning on igale javanistile teejuhiks siin keerulises digimaailmas. Meie tunnistaja Java, Me Jumal, on Meile andnud kõik siin ilmas ning kogumit lugedes saame Temaga lähedasemaks. "Java tunnistused" jaotuvad seitsmeks peatükiks, mis annavad meile aimu nii Meie maailma tekkest kui õigest ja valest, heast ja halvast. Allolevad peatüki lühikirjeldused annavad Meie veebilehe külastajale aimu "Java tunnistused" sisust.

Väljavõtteid pühakirjast

1. Java programmeeris oma kosmilise arvutiga Meie maailma.

2. Kõik inimesed, Jaava lapsed, sattusid siia tänu jumalikele koodiridadele.

3. Kosmiline peamälu on oma mahus igavesti suurenev - see annab Javale võimaluse meie Maailma kiiresti täiendada.

4. Kui Java aetakse raevu, toimub maailmas palju kurja, sest jumalikus koodis esineb siis bug'isid.

5. Java süntaksi üle kiruda on ülim patt ja eriti patused uskmatud kirjutatakse tulevaste uuendustega Maailma programmist välja.

6. Kõige üllamad Stack Overflow'i panustajad saavad Java kinnitusel pühakuteks.

7. Peale surma saavad truud Javanistid salvestatud kosmilistesse arhiividesse.
